# 田径数据收集

[**简体中文**](./README.md) |[ English](README.en.md)

![./Assets/博尔特.png](./Assets/博尔特.png)

本项目的几个**目标**：

1. 建立一些**现役的**知名运动员的主页（**[田联官方](https://worldathletics.org)**有主页，但我想让其更加**直观**）
2. 收集钻石联赛、奥运会、世锦赛的比赛数据（大概就是**成绩单**吧）
3. 对各个赛季一些项目的数据进行**总结**
4. **退役运动员**的主页也在计划之中
